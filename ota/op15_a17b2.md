# Android 17 Beta 2 — OnePlus 15

Requires OxygenOS **16.0.3.503** or **16.0.5.700**. If you're on a different version, use the rollback/upgrade packages at the bottom first.

Carrier devices (TMO/VZW) are not supported.

## Flash via Local OTA

### EU / IN

1. Copy the zip to internal storage.
2. Enable developer mode: **Settings → About device → Version → tap Build number 7×**.
3. **Settings → About device → Up to date → ⋮ → Local install** → select the zip → Extract → Upgrade.
4. Reboot.

### NA

1. Copy the zip to internal storage.
2. Install the [local update APK](https://oxygenos.oneplus.com/OTA-V-localUpdate-Oplus_key.apk).
3. Open the app → gear icon → select the zip → wait → reboot.

> If "Local install" is greyed out: disconnect all networks, then **Settings → Apps → App management → ⋮ → Show system → "Software update" → Storage usage → Clear data**. Retry with networks still off.
